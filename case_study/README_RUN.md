# How to Run

## Environment Setup

**Python version**: 3.9+ (tested on 3.12.3)

```bash
# Create and activate virtual environment
python -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate   # Windows

# Install dependencies
pip install -r requirements.txt
```

## Reproduce Results

**Option A — Command line:**
```bash
# Run EDA notebook
jupyter nbconvert --to notebook --execute eda.ipynb --output eda.ipynb

# Run modelling notebook (also generates conclusion_slide.pdf)
jupyter nbconvert --to notebook --execute model.ipynb --output model.ipynb
```

**Option B — Interactive (recommended):**
```bash
jupyter lab
# Open eda.ipynb and model.ipynb, then Run All Cells
```

## Files Overview

| File | Description |
|------|-------------|
| `utils.py` | Helper functions (docstring added, fixed for pandas 2.x) |
| `PR_COMMENTS.md` | Code review comments for utils.py |
| `eda.ipynb` | Exploratory data analysis |
| `model.ipynb` | Forecasting models, evaluation, and feature importance |
| `conclusion_slide.pdf` | One-slide executive summary (generated by model.ipynb) |
| `demand.csv` | Input: daily demand data |
| `promotions.csv` | Input: promotion start dates |
| `requirements.txt` | Python dependencies |

## Key Results

| Model | MAE | WAPE |
|-------|-----|------|
| Naive (lag-8) | 23.95 | 4.58% |
| Seasonal Naive (lag-52) | 28.72 | 5.49% |
| ETS (additive trend) | 20.28 | 3.82% |
| Ridge (per-group) | 20.39 | 3.80% |
| **Ridge (global)** | **16.31** | **3.04%** |
| GBR (global) | 18.37 | 3.42% |

Best model: **Ridge (global)** — 3.04% WAPE, a 34% improvement over naive baseline.
